用法：

In HTML Template Binding： {{ filter_expression | filter:expression:comparator }}

In JavaScript： $filter('filter')(array, expression, comparator)

expression 可以是string，object或function；comparator 为function，true或false(undefined)

expression 是string的情况
Search:
NamePhone
{{friend.name}} {{friend.phone}}
expression 是object的情况
Any:
Name only
Phone only
设置了comparator为true表示全字符匹配，设为false或undefined为模糊匹配。
Equality
NamePhone
{{friend.name}} {{friend.phone}}
