Enter number:
Default formatting: {{val| number}}
No fractions: {{val| number:0}}
Negative number: {{-val| number:4}}
        看下面几注数字：
        {{ 1234 | number}}
        {{ 1234.56 | number}}
        {{ .56 | number}}
        {{ 01.56 | number}}
        {{ '01e.56' | number}}
      
        结果：
        {{ 1234 | number}}
        {{ 1234.56 | number}}
        {{ .56 | number}}
        {{ 01.56 | number}}
        {{ '01e.56' | number}}
