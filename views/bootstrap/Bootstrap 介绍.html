Bootstrap 3.0 介绍

Bootstrap 3.0的新变化如下：

  • 趋向扁平化设计和提供更多可选主题
  • 移动优先和响应式设计，更好地支持移动端设备的开发
  • 全新定制：重新设计，用浏览器代替Heroku对其进行编译，更好的依赖支持、内置错误处理等。
  • 更好的box模型
  • 超强的网格系统：很好地支持手机、平板电脑、台式机和大屏幕布局。
  • 新的Glyphicons ico字体
  • 修复navbar组件
  • 让Modals更加响应式
  • 添加新组件：最新的混合panel和列表组
  • 删除一些组件，如accordion、submenus、typeahead及一些小组件，后期会根据需要添加一些新的进去。
  • 更加一致的基础和sizing类
  • 文档更加全面
  • 不再支持IE 7和Firefox 3.6：在IE 8中需要引用Respond.js来支持所有的媒体查询，关于浏览器兼容情况，你可以阅读这份文档。

关于Bootstrap 3发布的更多详情，可以查看发布说明。
