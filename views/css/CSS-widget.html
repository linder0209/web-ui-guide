各种箭头样式

箭头样式一般都是通用的，以下是其中的一种实现，主要利用border来实现，该实现参考bootstrap，兼容各大浏览器。下面分别给出css和less的实现

基于css的箭头代码

示例

  • 向左箭头
  • 向右箭头
  • 向上箭头
  • 向下箭头

代码

        <style type="text/css">
          .arrows-left, .arrows-right, .arrows-top, .arrows-bottom {
            display: inline-block;
            width: 0;
            height: 0;
            margin-left: 2px;
            vertical-align: middle;
            content: "";
          }
          .arrows-left{
            border-color: transparent #000000 transparent transparent;
            border-style: solid solid solid dotted;
            border-width: 5px 5px 5px 0;
          }
          .arrows-right{
            border-color: transparent transparent transparent #000000;
            border-style: solid dotted solid solid;
            border-width: 5px 0 5px 5px;
          }
          .arrows-top{
            border-color: transparent transparent #000000;
            border-style: dotted solid solid;
            border-width: 0 5px 5px;
          }
          .arrows-bottom{
            border-color: #000000 transparent transparent transparent;
            border-style: solid solid dotted;
            border-width: 5px 5px 0;
          }
        </style>         
        <ul>
          <li>向左箭头 <span class="arrows-left"></span></li>
          <li>向右箭头 <span class="arrows-right"></span></li>
          <li>向上箭头 <span class="arrows-top"></span></li>
          <li>向下箭头 <span class="arrows-bottom"></span></li>
        </ul>

定制化：我们可以定制箭头的颜色，大小和形状，下面以向左箭头为例来定制

  • 颜色的定制：很简单，只需设置border颜色即可 
                  .arrows-left-blue.arrows-left{
                 border-right-color: blue;
              }
    向左箭头(蓝色)
  • 大小的定制： 
                  .arrows-left-large.arrows-left{
                 border-width: 10px 10px 10px 0;
              }
    向左箭头(大小为10px)
  • 形状的定制：箭头默认是一个底边是高2倍的等腰三角形，我们可以通过设置border的宽度把他变成其他等腰三角形（包括等边三角形）的箭头 
                  .arrows-left-shape.arrows-left{
                border-right-width: 10px;
              }
    向左箭头(底边和高都为10的三角形)

标签页 导航

标签页导航有多种实现

常见的标签页

我们用 ul、li 和 a 三种元素实现导航标签页，首先给li设置margin-bottom: -1px，然后给活动的标签设置背景色，这样活动的标签底边框就不会显示。 代码如下：

<!DOCTYPE HTML>
<html>
  <head>
    <title>tabs 标签页</title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width">
    <style type="text/css">
      body{
        font-size: 12px;
        font-family: Helvetica,​Arial,​sans-serif;
      }
      .nav-tabs{
        list-style: none outside none;
        padding-left: 8px;
        border-bottom: 1px solid #DDDDDD;
      }
      .nav-tabs:before,
      .nav-tabs:after {
        content: " ";
        display: table;
      }
      .nav-tabs:after {
        clear: both;
      }
      .nav-tabs > li{
        float: left;
        margin-bottom: -1px;
      }
      .nav-tabs > li > a {
        text-decoration: none;
        display: block;
        margin-right: 10px;
        padding: 8px 15px;
        border-radius: 4px 4px 0 0;
        color: #3A7AB6;
        border: 1px solid #DDDDDD;
        border-bottom-color: transparent;
      }
      .nav-tabs > li > a:hover,
      .nav-tabs > li > a:focus{
        background-color: #EEEEEE;
        border-bottom-color: #DDDDDD;
      }
      .nav-tabs > li.active > a {
        background-color: #FFFFFF;
        color: #555555;
        font-weight: 700;
        cursor: default;
        border-bottom-color: transparent;
      }
    </style>
  </head>
  <body>
    <ul class="nav-tabs">
      <li><a href="#">Html</a></li>
      <li class="active"><a href="#">Css</a></li>
      <li><a href="#">Javascript</a></li>
    </ul>
  </body>
</html>
