nodejs的安装与配置

这里只简单的介绍在window环境下的安装与配置。

1、nodejs的安装

首先到官方 http://nodejs.org/下载最新版本，这里我们下载msi格式的安装文件，免去环境变量的配置， 我们也可以到 http://nodejs.org/dist/ 下载其他版本，下载后直接安装即可。

安装的时候选择默认设置，默认设置会安装npm（NPM的全称是Node Package Manager）。我本地安装的位置为：D:\Program Files\nodejs

安装完成后，nodejs会自动设置环境变量path：D:\Program Files\nodejs\ 。这里还需要手动设置一下环境变量 NODE_PATH： D:\Program Files\nodejs\node_modules ， 该变量的设置可以在需要用到的地方直接用NODE_PATH即可。

2、NPM的安装

如果在安装nodejs时，没有选择安装npm时，可以执行以下命令来下载npm文件（假设下载到目录D:\node\npm下面）。

git clone --recursive git://github.com/isaacs/npm.git

下载成功后，执行以下命令即可完成npm的安装，这时打开nodejs安装目录时会发现已经成功安装了node_modules。

cd D:\node\npm
node cli.js install npm -gf
npm install underscore

分别运行完以上命令后，会返回以下内容

underscore@1.4.3 node_modules\underscore

说明安装成功。

3、NPM相关插件的安装

默认情况下，NPM会安装系统提供的一些插件，比如request等。如果用户需要安装其他第三方插件时，需要执行以下安装命令。这里需要说明一下：如果用户把nodejs安装在c盘下，在安装npm相关插件到nodejs安装目录下时，这时需要用管理员身份运行cmd，即找到cmd.exe所在的目录，右键该文件，以管理员身份运行。
